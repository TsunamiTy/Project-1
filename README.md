# Pokemon-API-Project
An application that uses a Pokemon API to gather and distribute Pokemon names and their Pokedex number.  

## Technologies Used
- HTML
- JavaScript
- jQuery
- CSS
- AJAX
- Google Fonts
- Pokemon API
